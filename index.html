Welcome!

We will be exploring machine learning models and how to compare them by utilizing the Iris dataset originally created by R.A. Fisher and provided by UCI's Machine Learning Repository.



But what is machine learning? Machine learning systems work by using data to generate a model from the features composing each observation. After a model is generated learning from the data we have, the model can make predictions on new observations. This is our main tasks, so we want our model to predict with the highest accuracy possible.



So what are these attributes that our model will learn? This Iris dataset is a collection of Iris plants. Our model will learn from petal and sepal lengths and widths.


What exactly will our model be predicting? Each observation will have attributes associated with each observation. The Iris dataset contains 3 different species in total:



  • Virginica
  • Vericolor
  • Setosa




Our goal then for our model is to learn how the lengths and widths of the petals and sepals indicate which species the particular observation relates to. Each species tends to have similar features, so if our model and learn the relationship, the model should be able to make accurate predictions for new observations.


Let's take a closer look at our dataset. We can hover over each observation to see the features for each.



Each bubble here represents a single observation. Our total dataset has 150 recorded observations, so there are 150 bubbles. Smaller bubbles indicate smaller lengths and widths, while larger bubbles represent larger lengths and width.



By clicking the By Species button we can see the separation between the species. We can see things a bit more clearly here. The Seota species tends to be a bit smaller in size, while Virginica tends to be larger, with versicolor tends to be somewhere in teh middle. This could be a useful pattern for our model!



By clicking the Data Split button, we can how we will break our dataset into different groupings:



  • The Train Set
    • will be used to train our model. These observations will be used to learn the relationships to make accurate predictions.
  • The Validation Set
    • will be used while the model is trained. It is to ensure that training is moving in the right direction, ideally more accurate as more data is used!
  • The Test Set
    • is what we'll test our trained model on. We'll need to test to see how accurate our model can perform on data our model hasn't seen before. The model will not know the true species of the observations during testing.


The term model has been a bit vague so far. What do we mean by this?



The type of model to be used for machine learning depends on the type of data your dataset is comprised of. Depending on this, certain models are more suitable than others in accomplish tasks. Let's review a few.


If the dataset is only comprised of binary datatypes such as -5, 10, 0.001, 10,000 and the like two models and prmarily used:



  • Logistic Regression
    • is used when the task is to provide continuous out. Consider trying to predict the price of a house. Perhaps it could range from $10,000 to $10,000,000, it can be many values.
  • Linear Regression
    • is used when the task is to provide a discrete output. Consider trying to predict if an imagine is a cat, dog, or cow. There are only a few values that can be output by the model.


If the dataset is only comprised of multi-class datatypes, these can come in the form of cats, dogs, and cows, but all have the same attributes such as weight, hair type, or claw type. In this case two models are again primarily used:



  • Logistic Regression
    • as explained before, best for continuous output.
  • KNN
    • or K Nearest Neighbors is used when you need to group observations and output only a few values. K is an integer that decides how many neighbors should be included when deciding if an observation belongs to a certain type of observation. It is a powerful model, we'll use this for our walkthrough.


If the dataset is only comprised of text datatypes, these can come in in a wide variety of forms. Typically the task is to define the part of speech from the grouping of letters then score them, to understand their meaning:



  • Linear Regression
    • can often be an effective way to accomplish the task
